B.E. Course Structure

First Year (FE)

  • Engineering Mathematics I: Calculus, differential equations
  • Basic Electronics: Circuits, semiconductors
  • Programming in C: Basic programming and logic building

Second Year (SE)

  • Data Structures: Stacks, queues, trees, graphs
  • Digital Logic Design: Boolean algebra, flip-flops
  • Object-Oriented Programming: Using C++/Java

Third Year (TE)

  • Operating Systems: Memory management, scheduling
  • Computer Networks: Protocols, TCP/IP, routing
  • Web Technologies: HTML, CSS, JS, backend basics

Final Year (BE)

  • Machine Learning: Supervised, unsupervised algorithms
  • Cybersecurity: Cryptography, attacks, defenses
  • Major Project: Final year capstone project
